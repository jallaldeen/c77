To install, modify, or update Visual Studio, you must run the Visual Studio Installer as an administrator. If you try to modify Visual Studio as a typical user, you'll get a User Account Control notice prompting you for admin credentials. For more information, see User permissions and Visual Studio. The following procedures assume you have an internet connection. For more about how to modify a previously created offline installation of Visual Studio, see: Update a network-based installation of Visual Studio Control updates to network-based Visual Studio deployments
